# Project Brief

## 프로젝트명
**HYB Opportunity AI**

## 한 문장 정의
온라인 마켓플레이스 상품을 수집·비교·평가해 수익 가능성이 있는 판매 기회를 탐색하는 AI 기반 커머스 분석 시스템.

## 해결하려는 문제
온라인 판매 기회를 사람이 직접 찾으려면 다음 작업이 반복된다.

- 여러 마켓플레이스에서 상품 탐색
- 상품명이 다른 동일·유사 상품 판별
- 판매가와 매입가 비교
- 수수료와 비용 계산
- 순이익과 ROI 판단
- 후보 저장 및 재검토

HYB Opportunity AI는 이 과정을 데이터 파이프라인과 평가 엔진으로 자동화하는 것을 목표로 한다.

## 현재까지 확인된 기술 방향
- Python
- VS Code
- Git / GitHub
- 가상환경 `.venv`
- 환경변수 `.env`
- eBay Sandbox Browse API
- SQLite 기반 데이터 저장
- 상품 수집, 그룹화, 기회 계산을 분리한 모듈 구조

## 주요 개념
### Collector / Marketplace
외부 마켓플레이스에서 상품 데이터를 수집한다.

### Product Normalization
마켓마다 다른 상품 데이터를 내부 공통 형식으로 변환한다.

### Product Matching
동일하거나 유사한 상품을 비교하고 그룹화한다.

### Opportunity Engine
매입가, 예상 판매가, 비용, 순이익, ROI, 점수를 계산한다.

### Orchestrator
검색부터 평가·저장·출력까지 전체 흐름을 조정한다.

### Database
상품, 검색 결과, 분석 결과와 후보 이력을 저장한다.

### Dashboard
사용자가 검색 결과와 판매 후보를 확인하는 운영 화면이다.

## 최종 성공 기준
단순히 API 결과를 출력하는 것이 아니라 다음 흐름이 안정적으로 동작해야 한다.

```text
검색 조건 입력
→ 복수 마켓플레이스 데이터 수집
→ 데이터 정규화
→ 유사 상품 매칭
→ 예상 수익 계산
→ 위험도 및 기회 점수 평가
→ DB 저장
→ 대시보드 표시
→ 정기 자동 실행
```
