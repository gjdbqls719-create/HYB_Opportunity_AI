# Roadmap

## Sprint 0 — Environment & Git
상태: 완료에 가까움

- Python / VS Code
- 가상환경
- Git / GitHub
- 프로젝트 이동과 복구
- 환경변수 기본 관리

## Sprint 1 — Foundation
상태: 대부분 진행

- 프로젝트 기본 구조
- 공통 Product 모델
- 설정 관리
- DB 기본 구성
- 실행 진입점

## Sprint 2 — Documentation & Architecture
상태: 대부분 진행

- README 재정비
- 아키텍처 원칙
- 개발 기록 체계
- Starter Pack
- 작업 규칙

## Sprint 3 — Opportunity Engine
상태: 다음 단계

- 비용 항목 표준화
- 순이익, 마진율, ROI
- 판매가 추정
- 위험요소
- 점수와 판정 이유
- 단위 테스트

## Sprint 4 — Marketplace Pipeline
- eBay 코드 정리
- Sandbox와 Production 환경 분리
- 추가 마켓플레이스 어댑터
- 데이터 정규화
- API 오류와 속도 제한 처리

## Sprint 5 — Product Matching
- 제목 정규화
- 브랜드·모델·용량·상태 추출
- 유사도 계산
- 잘못된 매칭 방지
- AI 보조 평가 검토

## Sprint 6 — Dashboard
- 검색 실행
- 후보 목록
- 상품 상세
- 계산 근거
- 필터와 정렬
- DB 기록 조회

## Sprint 7 — Automation
- 정기 검색
- 작업 큐
- 실패 재시도
- 로그와 알림
- 후보 변화 추적

## Sprint 8 — Production Readiness
- 실제 API 키
- 약관 및 정책 검토
- 보안
- 배포
- 모니터링
- 비용 관리

## Alpha 완료 기준
- 하나 이상의 실제 또는 현실적인 데이터 소스
- 검색부터 DB 저장까지 한 번에 실행
- 계산 근거가 보이는 기회 평가
- 대시보드에서 결과 확인
- 핵심 엔진 테스트
- 반복 실행 시 데이터가 무너지지 않음

## Beta 완료 기준
- 복수 데이터 소스
- 정기 자동 실행
- 오류 복구
- 사용자 설정
- 실제 운영 데이터를 통한 정확도 보정
